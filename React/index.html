Installing React

Open terminal and type npm install create-react-app -g This step has to be done only once to have React on our computer globally for each new project we create

The next step is to create project. Run the command create-react-app some-project-name Once created navigate to that folder and there run the command npm start

State hook

State hook is giving us the ability to have state in functional components instead of class components. useState method can be called multiple times so it is best practice to use each type of state(like object, array, number, string etc.) in different useState because calling set state all other properties will be deleted.
